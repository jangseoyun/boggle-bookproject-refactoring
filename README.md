- 주요 내용
  - 메인 감정태그별 음악 선정 및 서평 랜덤 플레이 기능
  - 로그인, 로그아웃, 회원가입
  - 서평 글쓰기 기능
  - 서평/플레이리스트 좋아요 기능, 책 북마크 기능
  - 서평 리스트 (좋아요순, 최신순)
  - 서평 리스트 페이징
  - 현재 플레이리스트에 서평 추가 및 삭제
  - 서평 키워드 검색
  - 작성자만 서평 수정, 삭제
